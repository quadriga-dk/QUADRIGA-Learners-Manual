# Technische Voraussetzungen

## Nutzung des Templates

Wir empfehlen die Nutzung der Template-Funktion in GitHub. Sie können auch eine
`.zip`-Datei herunterladen und von dort aus beginnen.

Passen Sie dann die Inhalte an. Insbesondere sollten Sie achten auf die
Einstellungen in `_config.yml` und das Inhaltsverzeichnis in `_toc.yml`.

Wollen Sie GitHub-Pages für die Veröffentlichung des Jupyter Books nutzen, so
müssen Sie einerseits die Funktionalität in den Einstellungen der Repositoriums
anpassen und die GitHub Action in
`.github/workflows/deploy-book-python-sparql-r.yml` anpassen, ggf. umbenennen
und aktivieren. Zudem muss das Repositorium normalerweise öffentlich sein.

Vorgaben und Empfehlungen des Datenkompetenzzentrums QUADRIGA zur Erstellung
von OER und zur Nutzung von Jupyter Book und diesem Template finden Sie in der
<a href="https://quadriga-dk.github.io/QUADRIGA-OER-Manual"
    target="_blank">Anleitung</a>.
